## Цветовая палитра сайта
Для сайта была использована палитра gruvbox dark hard, тёмные цвета для фона и более яркие для текста и обводки/границ: 
base00: "1d2021" # ---- (фон)
base03: "665c54" # - (обводка)
base04: "bdae93" # + (мелкий текст)
base07: "fbf1c7" # ++++ (текст)

## Текст
Для всего сайта был использован шрифт UnifontEX в woff2 формате, который является красивым и очень хорошо читаемым, для хедеров была использована жирная версия шрифта, для ссылок на внешние ресурсы - версия с подчёркиванием.

## Ссылки
Ссылки имеют цвет обычного текста и при наведении меняют свой стиль на жирный. Ссылки на внешние ресурсы имеют подчёркивания, ссылки на другие страницы - нет (ну не считая той, которая отображает активную сейчас страницу)

## Списки
Списки на сайте присутствуют, но все их стилистические особенности отсутствуют

## Изображения
На главной странице используется .gif изображение для фона и для лого под пиксилезированным рендером, для страницы обо мне используется .png картинка.
